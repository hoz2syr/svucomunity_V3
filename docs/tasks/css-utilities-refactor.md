# مهمة: ترحيل تعويضات الألوان يدوية → `@theme` في Tailwind v4

**التاريخ:** 2026-06-24  
**الهدف:** إزالة ~36 كلاس CSS يدوي لت-defined colors من `src/index.css` ونقل قيمها إلى `src/styles/design-tokens.css` عبر `@theme` block  
**النطاق:** ملفات CSS فقط — لا تغيير في markup (`.tsx`)  
**النتيجة المتوقعة:** CSS أصغر (~120 سطر بدلاً من 362)، اكتشاف تلقائي للألوان في IDE، إزالة ازدواجية القيم

---

## الخطوات

### الخطوة 1: تحديث `src/styles/design-tokens.css`
- إضافة جميع الألوان المستخدمة يدوياً إلى `@theme` block

### الخطوة 2: تنظيف `src/index.css`
- حذف الأسطر 249–284 (color utility overrides)
- حذف الأسطر 69–320 (.btn-* component classes — غير part من التقرير لكنها مرشحة)
- الاحتفاظ بـ: animations/atoms (.animate-typewriter, shimmer-sweep, skeleton-*), input-field, glass-card, exam-container

### الخطوة 3: التحقق
- تشغيل `npm run build` والتأكد من نجاحه
- تشغيل الاختبارات (`npm test`) والتأكد من نجاحها
- التحقق من أن CSS الناتج لا يحتوي على التعريفات المحذوفة (كلمة كلاسassets )
