## Reporting a Vulnerability

To report a security vulnerability, please use the GitHub Security Advisory ["Report a Vulnerability"](https://github.com/hozai/svu-community/security/advisories/new) tab.

The SVU Community team takes all security reports seriously. We will acknowledge receipt within 3 business days and aim to provide an initial assessment within 10 business days.

Please **do not** report security vulnerabilities through public GitHub issues, discussions, or pull requests.

## Security Policy

- Encrypt all sensitive data in transit (TLS 1.2+) and at rest.
- Rotate all credentials immediately if exposed.
- Follow the principle of least privilege for all service accounts.
