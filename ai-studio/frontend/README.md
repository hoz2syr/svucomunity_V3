# OCR Frontend

React + TypeScript + Tailwind CSS interface for the OCR system.

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Features

- Drag & drop image upload
- Real-time job status tracking
- Download results in Markdown, PDF, or Word format
- Arabic RTL support
- Responsive design

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Axios
- Lucide React icons
